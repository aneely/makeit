## Key Learnings

### hs.execute uses a minimal PATH
Binaries called via `hs.execute` need full paths rather than bare command names.
Documented in PLAN.md under Capability Layers.

### hs.timer.waitUntil is async
Sets up a timer in the Hammerspoon daemon and returns immediately. `hs -c` exits before
the callback fires. This is expected behavior — apps launch in the background after the
condition resolves.

### All live testing has been against already-running apps
Cold-start (restart recovery) scenario not yet validated. Captured as
`cold-start-validation` story.

### DEFAULT_CONFIG_DIR as a test-override pattern
Introduced to let BATS tests exercise the "default path" branch without touching
`~/.config/makeit`. Generalizable pattern for future Makefile tests.

### direnv expands ~ in .envrc assignments
No special handling needed in `bin/makeit` — by the time `$MAKEIT_CONFIG` is read,
the path is already fully expanded.

### Glide path reference doc is not version-controlled in any project
Reference it in agent sessions at natural trigger points (sharing the tool, --version
flag work, Homebrew tap).
